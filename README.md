# agent-harness-course

《从零构建 AI Agent 与 Harness Engineering 实战课程》静态课程站点。

## 仓库结构

- `index.html`：单文件课程网站（CSS/JS 内联）
- `agent-harness-course.md`：课程 Markdown 原稿
- `README.md`：项目说明

## 本地预览

直接用浏览器打开 `index.html` 即可，无需构建步骤。

## GitHub Pages 部署

1. 推送 `main` 分支
2. GitHub Settings → Pages
3. Source 选择 `Deploy from a branch`
4. Branch 选择 `main`，目录选择 `/ (root)`
5. 访问：`https://<username>.github.io/agent-harness-course/`

## 贡献

优先在 `agent-harness-course.md` 维护内容，再同步到 `index.html`。
